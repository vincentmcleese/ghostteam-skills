## Process

1. Read the full transcript to identify main topics and structure
2. Extract 6-10 key topics for the breakdown section
3. Write a compelling 2-3 sentence hook
4. Generate timestamps based on topic transitions in transcript
5. Add closing statement with relevant call-to-action
6. Assemble all sections in order

## Guidelines

- Keep topic descriptions concise (1-2 sentences max)
- Timestamps should reflect natural content breaks
- Opening hook should create curiosity and communicate value
- Closing statement should emphasize timeliness or opportunity
- Match the energy and style of the video content
